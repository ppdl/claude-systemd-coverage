# 프로젝트 작업 지침

## 문서화 규칙

작업 중 새로운 단계를 수행하거나 문제를 해결할 때마다 아래 파일을 즉시 업데이트한다:

- **`docs/coverage-measurement-guide.md`**: 전체 과정의 단계별 재현 가이드.
  에이전트가 대화 없이도 처음부터 재현할 수 있도록 명령어, 결과, 발생한 문제와 해결 방법을 모두 기록한다.

업데이트 타이밍:
- 새 명령어를 실행하고 결과를 확인한 직후
- 오류가 발생하고 해결책을 찾은 직후
- 단계가 완료되어 체크박스를 닫을 수 있을 때

## 작업 방식

- 코드 탐색이나 에이전트 실행 전에 반드시 사용자와 계획을 먼저 논의한다.
- sdb 관련 작업은 `/sdb` 커스텀 슬래시 커맨드를 참조한다.
